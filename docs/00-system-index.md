# System Index - Master ID Registry

This file serves as the strict, single source of truth for all UI entities extracted from the Binance APK. Every screen, component, overlay, and widget documented in the 54 `docs/` files MUST be registered here with a unique ID format.

## 1. Screens (SCR)
*Containers, fragments, and activities.*

## 2. Shared Components (COMP)
*Reusable UI modules (Rows, cards, badges).*

## 3. Overlays & Dialogs (OVR)
